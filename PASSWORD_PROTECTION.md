# הגנת סיסמה לאתר

האתר מוגן בסיסמה בסיסית. כדי לגשת לאתר, יש להזין את הסיסמה במסך ההתחברות.

## סיסמה נוכחית

הסיסמה מוגדרת בקובץ `src/components/Login.tsx` בשורה:
```typescript
const SITE_PASSWORD = "idf_2025";
```

## שינוי הסיסמה

כדי לשנות את הסיסמה:

1. פתח את הקובץ `src/components/Login.tsx`
2. שנה את הערך של `SITE_PASSWORD`:
   ```typescript
   const SITE_PASSWORD = "הסיסמה_החדשה_שלך";
   ```
3. שמור את הקובץ

## איך זה עובד

- הסיסמה נשמרת ב-`sessionStorage` (נמחקת עם סגירת הדפדפן)
- האימות תקף ל-24 שעות ממועד ההתחברות
- כל טאב חדש ידרוש הזנת סיסמה מחדש
- אין צורך ב-Supabase Auth או אימות מורכב

## אבטחה

⚠️ **חשוב לדעת:**
- הסיסמה נשמרת בקוד הקליינט (לא מומלץ לייצור)
- זהו פתרון בסיסי להגנה מפני גישה אקראית
- עבור אבטחה גבוהה יותר, מומלץ להשתמש ב-Supabase Auth

## אפשרויות שיפור עתידיות

אם תרצה להפוך את זה למאובטח יותר:
1. העבר את הסיסמה ל-variable סביבה
2. השתמש ב-Supabase Auth עם משתמשים אמיתיים
3. הוסף rate limiting למניעת brute force
4. הוסף 2FA (אימות דו-שלבי)
